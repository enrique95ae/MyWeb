Health Web App

  • Technologies used: Angular, HTML and CSS
  • Features: Responsive, API integration, PDF generation, reusable and scalable
  • Backend: Custom made Health Web API specifically for this project

This project exemplifies my knowledge of the entire SDLC cycle. Along with the API this angular website constitutes a Full Stack full scope project. I have based my efforts in applying modern design patterns, coding, naming and organizing conventions. I have also taken into consideration the UI/UX face of web projects making not only an easy on the eyes web but also very intuitive.
